# Contract Templates

One contract template per note type. Each contract is split into:

- **Pre-draft specification** (filled before drafting, binding)
  - Phase 1: scope commitment (from registry context)
  - Phase 2: execution plan (requires mathematical reasoning)
- **Post-draft integration record** (filled after drafting, evidential)

See `docs/design/methods/formal-contracts.md` for the full contract framework.
